# Marketsynth AI Context

Official AI governance and context repository for Marketsynth.

This repository is the Source of Truth for AI-assisted development in Cursor, ChatGPT, Claude Code, Codex and other engineering agents.

## Purpose

- Preserve project constitution, architecture and runtime invariants.
- Provide Cursor with stable development context.
- Prevent architectural drift.
- Separate frozen project decisions from temporary chat discussion.
- Maintain a weekly audit trail.

## Document Status

- `FROZEN` — foundation, changed only through formal architectural audit.
- `ACTIVE` — current and authoritative.
- `DRAFT` — under discussion, not implementation authority.
- `SUPERSEDED` — replaced by newer document.
- `DEPRECATED` — preserved for history only.
