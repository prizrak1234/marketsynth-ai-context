# Approval Boundary

Status: ACTIVE

## Approval is not Readiness

Readiness means the artifact appears technically executable.

Approval means a human explicitly allowed execution.

## Mandatory Approval Areas

- external publishing;
- API calls that mutate external systems;
- production data changes;
- payment/budget operations;
- account or credential changes;
- user-visible messaging.

## Approval Expiry

Approvals should be scoped and time-bound where appropriate.
