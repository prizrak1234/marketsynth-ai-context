# Agent Runtime

Status: ACTIVE

## Agent Role

Agents operate inside explicit domain boundaries.

They may draft, analyze, route, summarize and prepare execution packages.

## Agent Limits

Agents must not:

- bypass owner approval;
- change frozen architecture;
- silently persist memory across tenant boundaries;
- run tools outside allowlist;
- infer unavailable authority.

## Programmer Domain

Programmer Agent initially produces technical task drafts.

Direct codebase mutation requires explicit implementation phase.
