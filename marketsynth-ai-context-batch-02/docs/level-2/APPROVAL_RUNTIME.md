# Approval Runtime

Status: ACTIVE

## Readiness

Readiness means technically prepared for execution.

Readiness is not permission.

## Approval

Approval means a human explicitly allows execution.

## Decisions

- pending;
- approved;
- rejected;
- expired.

## Owner Unavailable

If owner is unavailable:

- do not assume approval;
- escalate;
- wait;
- expire;
- or follow configured delegation policy.
